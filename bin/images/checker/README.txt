CHECKER PIECE IMAGES NEEDED
===========================

Place the following 4 PNG images in this folder:

1. red-piece.png      - Regular red checker piece
2. red-king.png       - Red king (crowned piece)
3. black-piece.png    - Regular black checker piece
4. black-king.png     - Black king (crowned piece)

Recommended specifications:
- Size: 80x80 pixels
- Format: PNG with transparent background
- Style: Simple, circular checker pieces
- Kings should have a crown or "K" marking

NOTE: The game will run without these images using placeholder graphics,
but adding proper images will greatly improve the visual experience.

Where to get images:
- Create your own in any image editor
- Use AI image generators (ChatGPT, DALL-E)
- Download from free game asset sites (OpenGameArt.org)
- Use simple colored circles with a crown for kings
